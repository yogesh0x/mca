# Skin Cancer Detection using CNN

## Project Overview
This project aims to build a deep learning model to detect skin cancer using a Convolutional Neural Network (CNN). A simple, interactive Streamlit GUI allows users to upload an image and receive predictions (Benign or Malignant).

---

## Technologies Used
- Python
- TensorFlow / Keras
- Streamlit
- Scikit-learn
- Matplotlib / Seaborn

---

## Dataset Details
**Dataset Name:** Melanoma Cancer Dataset  
**Structure:**
```
melanoma-cancer-dataset/
├── train/
│   ├── benign/
│   └── malignant/
├── test/
    ├── benign/
    └── malignant/
```
**Classes:**
- Benign (non-cancerous)
- Malignant (cancerous)

Images are resized to 224x224 for input to the CNN.

---

## Project Files
### 1. `train_model.py`
- Preprocesses the dataset using `ImageDataGenerator`
- Builds a CNN with 3 convolutional blocks
- Trains the model using `fit()`
- Saves the trained model as `skin_cancer_cnn.h5`

### 2. `app.py`
- Loads the trained model
- Provides a GUI to upload an image
- Classifies the image as Benign or Malignant
- Displays the image and prediction label

---

## Model Architecture
```python
model = Sequential([
    Conv2D(32, (3,3), activation='relu', input_shape=(224,224,3)),
    MaxPooling2D(2,2),
    Conv2D(64, (3,3), activation='relu'),
    MaxPooling2D(2,2),
    Conv2D(128, (3,3), activation='relu'),
    MaxPooling2D(2,2),
    Flatten(),
    Dense(512, activation='relu'),
    Dropout(0.5),
    Dense(1, activation='sigmoid')
])
```

- Loss Function: Binary Crossentropy
- Optimizer: Adam
- Evaluation Metric: Accuracy

---

## Training Parameters
- Image Size: 224x224
- Batch Size: 32
- Epochs: 30
- Callbacks: `EarlyStopping`, `ReduceLROnPlateau`

---

## Streamlit App Workflow
1. Upload an image (`.jpg` / `.png`)
2. Model loads and processes the image
3. Displays the uploaded image with predicted label
4. Outputs:
   - Benign (non-cancerous)
   - Malignant (cancerous)

---

## Example Results
**Validation Accuracy:** ~85-90% (depends on dataset size)  
**Confusion Matrix, Classification Report** are printed after training.
